# دورة SQL 101 — Satr (Tuwaiq Academy)

هذا المستودع يحتوي على مشروع لمادة دورة (SQL 101) التي أنجزتها عبر منصة سطر التابعة لأكاديمية طويق.

## محتويات المستودع
- `MySQL 101 Course - Track.docx` — ملف المشروع بصيغة وورد.
- `MySQL 101 Course - Track.pdf` — ملف المشروع بصيغة PDF.

## كيفية العرض
1. لعرض المشروع بسهولة، افتح الملف `MySQL 101 Course - Track.pdf`.
2. يمكن أيضًا تحميل الملف وفتحه محليًا على أي قارئ PDF.

## ملاحظات
- قاعدة البيانات أنشئت باستخدام MySQL Workbench.
- يحتوي المشروع على مثال لقاعدة بيانات (شركة)، ومثال لقاعدة بيانات (مدرسة) تحتوي الجداول (الطلاب، المعلمين، المواد) مع عمليات الإدخال والتعديل والعرض المطلوبة.
- تم توثيق جميع الخطوات بلقطات شاشة ضمن الملف المرفق.
